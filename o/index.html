How it works

Solid Chat is a decentralized Chat, build on top of Solid. No central sever is required to run or set up a chat. All personal data about the chat is stored on your POD. Requests to join and updates of a chat are sent to the inbox of your opponent directly.

The source of this app can be found on Github.
